The knowledge manifold hero; a data-driven wireframe surface where height is Performance, holes are gaps, and footprint tracks exam weight.

```jsx
<Manifold width={828} height={540} scale={182} glow={0.7} />
<Manifold width={353} height={330} scale={112} grid={56} lineWidth={0.65} />  // phone thumbnail
```

Pass a custom `surface` built from live stats to reshape it. Amber to lilac to blue runs across the screen axis; never reuse those hues decoratively elsewhere. Production: Three.js displaced plane, D3 contour fallback, same `surface` structure.
